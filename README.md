# kloud-moneywash
 QBCore Configurable Moneywash

# Features
- Zones or Target function (ox_lib / ox_target or qb-target)
- Notify and DrawText (ox_lib or qb-core)
- Per Item Duration
- Metadata Usage
- Tax
- Clean Animation
- Easily Add Locations
- Compatible with ox_inventory / qb-inventory (not tested)
